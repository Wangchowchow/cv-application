# CV Application
This is a CV application web page.
## Created With
* HTML
* CSS
* Javascript
* React
## Getting started
```
git clone https://github.com/wangchowchow/cv-application.git
cd cv-application
npm install
npm start
```
## Notes
* Two modes available by toggle.
* Mode: Input to edit CV.
* Mode: Preview to hide buttons in page.
